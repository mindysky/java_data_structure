# Java Project

#### 开发环境检查

- jdk 1.8

​    配置：JAVA_HOME、path

- maven 3.6

​    配置：MAVEN_HOME、本地仓库路径、中央仓库地址、jdk版本

- ideaIU-2020.2.3：

 注意：在idea的欢迎页面选择Configure > settings

​    配置：Java Compiler、File Encodings、Maven、Auto Import、Code Completion > Match case（取消）

​    插件：lombok、MyBatisX
